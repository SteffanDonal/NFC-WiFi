# NFC-WiFi
NFC reader with WiFi 
This project joins a NFC reader(PN532) to a WeMos D1 Mini to allow for WiFi access.

Want one? Sign up to preorder one now: https://goo.gl/forms/MPRosQnBkixg8Vhj2

# Require Hardware
* PN532 Board
* WeMos D1 Mini 4M
* PCB to join the two
* Enclosure

# Pinout Notes

## Led (APA102)
	* data 5
	* clock 4

## D1 Mini (WeMos 4M)
	* sck - D0 - 16 
	* miso - D5 - 14
	* mosi - D6 - 12
	* ss - D7 - 13
